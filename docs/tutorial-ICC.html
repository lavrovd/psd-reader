ICC

psd-reader does not extract or apply ICC to the bitmap.

Lack of such processing may affect the resulting colors in various degrees depending on what color profile was used when creating the PSD file, in particular it will affect CMYK files.

For general use this is typically not an issue, but you need to be aware of this if you intend to use the loaded PSD for print of professional photography.
